# MEMS – How to Run

## Compile

From the project root (where `Main.java` is):

```bash
Run interactively

javac *.java
java Main

Then type commands at the prompt, for example: c – create ensemble a – add musician se – show current ensemble sa – show all ensembles u – undo r – redo x – exit

Run all tests (Linux/macOS)
./run_tests.sh
Outputs go into test_outputs/.


Run all tests (Windows)
In Command Prompt:
run_tests.bat
Outputs go into test_outputs\.
```
